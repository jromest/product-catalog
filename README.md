# Simple Product Catalog App

This app is made with:

- [React](https://reactjs.org/)
- [Reach Router](https://reach.tech/router/)
- Styled with [Reactstrap](https://reactstrap.github.io/)
- Build with [Vite](https://vitejs.dev/)
